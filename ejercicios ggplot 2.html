1 Introducción

Este documento presenta un análisis exploratorio de datos utilizando ggplot2. El objetivo es visualizar las relaciones entre variables e interpretar los patrones observados en los datos.

Código 
library(ggplot2)
library(tidyverse)
library(readr)
Código 
students <- data.frame(

id = 1:20,

hours_study = c(2,3,4,5,6,3,4,7,8,5,6,7,3,2,9,10,4,6,8,5),

score = c(50,55,65,70,75,60,66,80,85,72,74,78,58,52,90,95,67,76,88,73),

gender = rep(c("Male","Female"),10),

major = rep(c("Engineering","Business","Arts","Science"),5)

)

2 Descripción del conjunto de datos

El conjunto de datos contiene información sobre:

  • Horas de estudio
  • Calificación obtenida
  • Género del estudiante
  • Carrera

El objetivo es analizar si existe una relación entre las horas de estudio y el rendimiento académico.

3 Ejercicio 1 – Diagrama de dispersión

3.1 Enunciado

Crear un diagrama de dispersión entre las horas de estudio y la calificación.

Código 
p <- ggplot(students, aes(hours_study, score)) +
geom_point()

print(p)

3.2 EJERCICIO 2 – Cambiar color de los puntos

Enunciado

Cambiar el color de todos los puntos a azul.

Código 
p <- ggplot(students, aes(hours_study, score)) +
geom_point(color="blue")

print(p)

3.3 Explicación

Este gráfico presenta la misma relación entre horas de estudio y calificación, pero ahora todos los puntos tienen un color azul uniforme.

4 Mejora de la gráfica

Código mejorado

Código 
p <- ggplot(students, aes(hours_study, score)) +

geom_point(color="blue", size=3)+

labs(

title="Horas de estudio vs calificaciones",

x="Horas de estudio",

y="Calificación"

)+

theme_minimal()

print(p)

4.1 EJERCICIO 3 – Color según género

Enunciado

Asignar color según el género.

Código 
p <- ggplot(students, aes(hours_study, score, color=gender)) +

geom_point(size=3)

print(p)

4.2 Explicación

El gráfico diferencia los estudiantes según su género mediante distintos colores.

Interpretación

Se observa que tanto hombres como mujeres siguen una tendencia similar entre horas de estudio y calificación.

Código 
p <- ggplot(students, aes(hours_study, score, color=gender)) +

geom_point(size=3)+

geom_smooth(method="lm")+

labs(

title="Relación entre horas de estudio y calificación por género",

x="Horas de estudio",

y="Calificación",

color="Género"

)+

theme_minimal()

print(p)

# Introducción

```{r}
library(ggplot2)
library(tidyverse)
library(readr)
```

``` {r}
students <- data.frame(

id = 1:20,

hours_study = c(2,3,4,5,6,3,4,7,8,5,6,7,3,2,9,10,4,6,8,5),
```
