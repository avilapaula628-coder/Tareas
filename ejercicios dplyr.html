1 Introducción

Este documento presenta soluciones a ejercicios de manipulación de datos utilizando el conjunto de datos gapminder y el paquete dplyr.

2 Bibliotecas requeridas

Código 
library(dplyr) 
library(gapminder) 
library(tidyr)

3 Ejercicio 1

3.1 Pregunta interpretada

Seleccione solo las variables country, year y life expectancy del conjunto de datos.

3.2 Solución

Código 
gapminder %>% 
select(country, year, lifeExp)

El resultado muestra solo el nombre del país, el año y la esperanza de vida.

14 Ejercicio 12

Enunciado interpretado

Encontrar la esperanza de vida máxima para cada continente.

14.1 Solución:

Código 
gapminder %>% 
group_by(continent) %>%
summarise(max_lifeExp = max(lifeExp), .groups="drop")
# A tibble: 5 × 2
  continent max_lifeExp
  <fct>           <dbl>
1 Africa           76.4
2 Americas         80.7
3 Asia             82.6
4 Europe           81.8
5 Oceania          81.2

14.2 Interpretación del resultado

El resultado muestra el país con mayor esperanza de vida registrado dentro de cada continente. Esto permite comparar los niveles máximos de desarrollo en salud entre regiones.

14.3 Explicación

La función group_by(continent) agrupa los datos según el continente para poder realizar cálculos dentro de cada grupo.

La función summarise() permite calcular estadísticas resumen.

La función max(lifeExp) obtiene el valor máximo de esperanza de vida dentro de cada continente.

El argumento .groups=“drop” evita que el resultado permanezca agrupado.

21 Ejercicio 19

22 Ejercicio 20

Enunciado interpretado

Identificar los 5 países con mayor crecimiento en GDP per capita entre 1952 y 2007.

22.1 Solución

Código 
gapminder %>% 
filter(year %in% c(1952,2007)) %>% 
select(country, year, gdpPercap) %>%
